# Projeto To-Do List - Faculdade

Sistema de gerenciamento de tarefas com autenticação

## Tecnologias
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: PostgreSQL

## Estrutura do Projeto
```

├── frontend/     # Aplicação React
├── backend/      # API Node.js
├── docs/         # Documentação das ACs
└── README.md
```
